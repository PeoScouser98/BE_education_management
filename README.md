## Usage

### 1. Install

```bash
$ npm install
```

### 2. Dev command

```bash
$ npm run dev
```

### 3. Build command

```bash
$ npm run build
```

### 4. Run application trên môi trường Production

```bash
$ npm start
```

### 5. Run redis

```bash
$ docker-compose up -d redis
```
